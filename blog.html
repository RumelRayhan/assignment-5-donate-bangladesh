Important FAQ

What is Document Object Model ?

The Document Object Model is a programming API for documents. The object model itself closely resembles the structure of the documents it models.

How do you select an element from the DOM ?

We can select an element from DOM by document.getElementById(),document.getElementsByClass(), document.querySelector(), document.querySelectorAll()

How do you add a new element to the DOM?

You can add a new element to the DOM using the createElement() and appendChild() methods.

How can you remove an element from the DOM?

To remove an element, use the removeChild method. First, get a reference to the parent element, then call removeChild
